### Usage

|         |                                                    |
| ------- | ---------------------------------------------------|
| Clone   | `git clone https://github.com/lvnhmd/advent-of-code`|
| Install | `npm i`                                             |
| Test    | `npm test`                                          |
| Build   | `npm run build`                                     |
| Run     | `npm start <mode> <filepath> `                      |

##### `npm start <mode> <filepath>`

| Mode    |  Filepath             |
| ------- | --------------------- |
| path    | data/directions.txt   |
| wrap    | data/dimensions.txt   |
